# 🛡️ Zero-Code-First-Framework: GateKeeper AI

**GateKeeper AI** is an advanced, phase-gated system engineering skill designed exclusively for **Claude**. It transforms standard conversational LLM interactions into a rigorous, autonomous software engineering pipeline.

> **The Philosophy:** Stop writing rushed, bug-prone code. Let the GateKeeper architect first, test continuously, and deploy with absolute certainty.

---

## 🧠 Why GateKeeper AI?

The biggest bottleneck when working with AI assistant tools is the immediate generation of messy, monolithic code that ignores existing project architecture. 

**GateKeeper AI** enforces a strict, multi-phase separation of concerns. It bars entry to immediate coding, forcing Claude to act as a meticulous Principal Systems Architect who reasons through engineering choices *before* touching the codebase.

### 🔄 The 4-Phase Gated Lifecycle

| Phase | Focus | Deliverable | Gated? |
| :--- | :--- | :--- | :---: |
| **1. Spec & Architecture** | Blueprinting & Deconstruction | File topology, API schemas, micro-task checklist | **YES** (Requires Approval) |
| **2. Modular Coding** | Implementation | Clean, DRY, production-grade decoupled modules | No |
| **3. Automated QA** | Stress Testing & Reasoning | Edge-case analysis, memory leak scans, security check | No |
| **4. Integration** | Deployment & Specs | `.env` mappings, setup scripts, testing commands | No |

---

## 🎯 Who is this for?

**GateKeeper AI** is built specifically for:
* **Software Engineers & Developers:** Who use Claude daily but are tired of spending hours fixing context-blind, rushed, or buggy AI code.
* **Tech Leads & Software Architects:** Looking to enforce a strict, standardized engineering workflow across their teams when leveraging AI generation.
* **Indie Hackers & Solopreneurs:** Who need a reliable, senior-level AI partner to architect and test scalable systems from scratch without losing control of the codebase.
* **Product Managers:** Who want to quickly turn high-level product requirements into granular technical specifications and architectural blueprints.

## ⚡ Quick Start (How to Add the Skill)

### Option 1: Claude Projects (Recommended)
If you are using Claude Pro or Claude Team, you can embed this system-level skill permanently:
1. Open your **Claude Project**.
2. Copy the entire contents of [`system-prompt.txt`](./system-prompt.txt) from this repository.
3. Paste it directly into the **Project Instructions** section.
4. Name the instruction block: `GateKeeper AI Core`.
5. Start prompting! Claude will now automatically engage the framework.

### Option 2: Direct Session Initialization
Simply copy the text inside `system-prompt.txt` and paste it as your very first message in any new Claude chat session.

## 💎 Key Features

* **Anti-Rushed Code Enforcement:** GateKeeper AI will flat-out refuse to write code on turn one. It ensures your system architecture is bulletproof first.
* **Edge-Case Resilience:** Built-in automated reasoning that stress-tests its own logic against race conditions, null payloads, and concurrency bottlenecks.
* **Incremental Slicing:** Perfect for large-scale applications; breaks massive tasks into unblocking, reviewable micro-deliverables.
